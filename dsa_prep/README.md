# DSA Interview Prep — MAANG / Top Product Companies

A curated, topic-organized set of the highest-frequency coding interview problems
(roughly the well-known "Top 150" tier — Blind 75 + NeetCode 150 extras), each with:

- `problem.md` — statement, constraints, approach/intuition, complexity, and **company tags**
- `solution.py` — clean Python solution + runnable sample test cases (`python3 solution.py`)

> **Note on company tags:** these reflect long-standing, publicly known "commonly asked at"
> associations for each problem (the kind LeetCode's community/company-tag pages have shown
> for years). Treat them as *priority signals*, not a guarantee of what any specific company
> is asking this week — company interview pools rotate.

## How to use this

1. Work topic by topic in the order below — it's roughly increasing difficulty/dependency
   (e.g. two pointers before sliding window, trees before graphs, 1-D DP before 2-D DP).
2. Read `problem.md` first, attempt it yourself, *then* check `solution.py`.
3. Run the solution file directly to sanity-check: `python3 <topic>/<NN>_<name>/solution.py`
4. See `TOP_LIST.md` for a flat priority-ordered checklist if you only have time for a subset.
5. See `COMPANY_INDEX.md` to filter by a specific company you're interviewing with.
